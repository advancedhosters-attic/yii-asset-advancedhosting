This package provides assets shareable between different AdvancedHosting projects:

- logos and css bundled with Yii asset bundle
- config settings
- translations

## Installation

The preferred way to install this yii2-extension is through [composer](http://getcomposer.org/download/).

Either run

```sh
php composer.phar require "advancedhosters/yii-asset-advancedhosting"
```

or add

```json
"advancedhosters/yii-asset-advancedhosting": "*"
```

to the require section of your composer.json.
